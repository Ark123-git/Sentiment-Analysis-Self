# Sentiment Analysis

A logistic regression model for classifying tweet sentiments after text preprocessing and TF-IDF vectorization.

## Description
The project includes:
- Text preprocessing using NLTK
- TF-IDF-based vectorization
- Logistic Regression model
- Achieved accuracy of 77.6 percent
- Model saved using pickle

## Features
- End-to-end machine learning pipeline
- Cleaned and stemmed text
- Train and infer scripts

## Technologies Used
Python  
NLTK  
Scikit-learn  
Pandas
